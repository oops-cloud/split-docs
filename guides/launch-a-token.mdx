<Steps>
  <Step title="Details">
    Name and ticker. 1B fixed supply, no creator allocation.
  </Step>
  <Step title="Branding">
    Image, description, website, Twitter, Telegram. All optional except image.
  </Step>
  <Step title="Launch">
    Optional initial buy in SOL. One wallet popup creates the token, initializes rewards, and registers you as first holder.
  </Step>
</Steps>

## Cost

About 0.02 SOL for creation. No platform fee. Initial buy is optional.

## Custom addresses

Every Split token gets a custom branded address ending in **SpLt**.
